# Luna Assistant
A responsive, Voice-activated AI assistant inspired by systems like JARVIS, built with Python and Flask.

## Features
* **Voice Recognition:** Listen for commands using 'SpeechRecognition'.
* **Text-to-speech:** Responds vocally using 'pyttsx3'.
* **Web interface:** Intergrated Flask backend for future dashboarding.
* **Secure:** Enviroment variables managed via '.env'(keys not included).

## Setup
1. Clone the repository.
2. Install dependencies: 'pip install -r requirements.txt'
3. Create a '.env' file and add your 'OPENAI_API_KEY'.
4. Run the assistant: 'python Luna_assistant.py'
